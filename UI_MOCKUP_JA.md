# CompanionDemoApp 画面イメージ（モック）

実機スクリーンショットがこの環境では取得できないため、
現在のSwiftUI実装に合わせた**画面モック**をテキストで示します。

---

## 1) Onboarding（初回設定）

```text
┌──────────────────────────────┐
│ ようこそ                      │
├──────────────────────────────┤
│ [AIの基本設定]                │
│ AI名                [Airi   ] │
│ 話し方              [friendly▼]│
│ テーマ色(HEX)       [#7C9DFF] │
│ 性格タグ(カンマ)    [聞き上手,励まし]│
│                                │
│        [ はじめる ]            │
└──────────────────────────────┘
```

## タブバー構成

```text
[Chat] [History] [Stats] [Customize]
```

- 初回起動: Onboarding → 完了後にタブ画面へ遷移
- 2回目以降: 保存データがあればタブ画面を直接表示
